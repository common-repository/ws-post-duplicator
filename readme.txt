== Installation ==

1. Download [WS Maintenance Mode](https://wordpress.org/plugins/ws-maintenance-mode/) plugin.   
2. Upload the WS Maintenance Mode plugin from your admin panel.    
3. Activate the WS Maintenance Mode plugin. 
4. Now you can see website in maintenance after logged out admin.
